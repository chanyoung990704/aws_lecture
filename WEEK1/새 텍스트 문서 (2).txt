클라우드 컴퓨팅이란 
	인프라를 하드웨어로 생각하지 않고 소프트웨어처럼 엑세스하고 사용
	이점
		자본 비용을 가변 비용으로 대체
		규모의 경제로 얻게 되는 이점
		용량을 추정할 필요 없음
		속도 및 민첩성 향상
		중요한 일에 집중
		몇 분만에 전 세계에 배포
	클라우딩 컴퓨팅 배포 모델
		클라우드 / 하이브리드 / 온프레미스 ( 프라이빗 클라우드 )

	AWS란
		다양한 글로벌 클라우드 기반 제품 제공하는 안전한 클라우드 플랫폼
		컴퓨팅, 스토리지, 네트워킹, 데이터베이스 등에 대한 온디맨드 엑세스 제공
		유연성 제공
		필요한 개별 서비스 사용 기간 동안만 비용 지불
		서비스들이 서로 유기적으로 작동
	AWS 글로벌 인프라
		글로벌 네트워크 성능을 갖춘 유연하고 안정적이며 확장가능하고 안전한 클라우드 컴퓨팅 환경 제공
		AWS 리전
			리전 간 데이터 복제는 사용자가 제어
			리전 간 통신은 AWS 백본 네트워크 인프라 사용
			각 리전 완전한 이중화 및 짧은 지연시간 
			세 개 이상의 가용 영역으로 구성
			데이터 거버넌스 / 법적 요구 사항 / 고객에 대한 근접성 / 리전 내 사용 가능한 서비스 / 비용 
		가용 영역
			AWS 인프라의 완전히 격리된 파티션
			99개 가용영역
			가용 영역은 개별 데이터 센터로 구성
			내결함성 갖게 설계
		데이터 센터
			보안을 위해 설계
			데이터가 상주하고 데이터 처리가 이루어지는 곳
	컴퓨팅 서비스
		EC2 / EC2 Auto Scaling / ECS / ECR / Elastic Beanstalk / Lambda / EKS / Fargate
