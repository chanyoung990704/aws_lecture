○클라우드 역할
	클라우드 엔터프라이즈 아키텍트
		비즈니스를 위한 클라우드 서비스 제공
		협업 통해 비즈니스 요구 사항 확보 / 솔루션에 독립적인 아키텍쳐 설계 / 비즈니스에 다양한 모델 제시 / 아키텍처 검증, 구체화 및 확장 / 모니터링 및 업데이트
	프로그램 관리자
		클라우드가 적절하게 관리되도록 할 책임
		운영 팀 관리 / 클라우드 지표 관리 및 모니터링 / 서비스 보고서 관리
	재무 관리자
		클라우드의 재무 통제 관리
		비용 코딩 담당 / 비용 분배 / 비용 사용량 파악 / 비용 최적화
	클라우드 인프라 아키텍트
		솔루션에 종속된 클라우드 인프라 아키텍처 설계 담당
		계획 개방 및 유지 관리 / 다른 팀과 협력
	클라우드 운영 엔지니어
		클라우드 인프라 및 공유 서비스의 구축, 모니터링 및 관리 담당
		클라우드 인프라 아키텍트와 협업
		서비스 요구 사항이 충족되는지 확인
	클라우드 운영 엔지니어 - 관리
		OS관리 / 패치 및 업데이트 관리 / 템플릿 관리 / 문서 변경 사항 / 클라우드 인프라 태깅 및 검토 / 용량 관리 / 가상 네트워크 관리 / 애플리케이션 복원력 관리
	클라우드 운영 엔지니어 - 지원
		클라우드 서비스에 대한 운영 지원 제공 / 성능 튜닝 / 인시던트 대응 및 에스컬레이션 / 원인 분석 / 설명서 검토 수정 / 백업 및 복구 지원 / 규정 준수 프로그램 모니터링 및 보고
	클라우드 보안 아키텍트
		보안 요구 사항 지정 담당
		다른 팀과 협업 / 보안 구성 체크리스트 설계 및 유지관리 / 위험 평가 계획 / 기업 보안 대책 설계 / 인시던트 대응
	보안 운영 엔지니어
		보안 관리, 모니터링 및 적용 담당
		기업 보안 정책 및 절차 구현 / 규정 준수 관리 및 적용 / 보안 구성 관리 / 보안 그룹 구성 / 취약성 테스트 및 분석 / 보안 평가
	애플리케이션 아키텍트
		클라우드에 최적화된 애플리케이션 설계 담당
		다른 팀과 협업 / 용량 및 확장성 요구 사항 수행 / 전문적인 소프트웨어 지식 제공
	애플리케이션 개발자
		애플리케이션 변경 관리 / 코드 릴리스 관리 / 코드 배포 관리 / 지원 제공 / 설명서 관리 / 최적화 기법 개발
	DevOps 엔지니어
		빠르고 확장 가능한 워크플로우 구축 및 운영
		다른 팀과 협업 / 자동화 솔루션 설계 및 구축 / 지속적인 구축, 통합, 배포 및 Infrastructure as Code 구현 / 운영 개선 사항 검토 및 권장 / 애플리케이션 테스트 및 복구 수행 / 변경 관리 프로세스 개발 및 유지 관리
